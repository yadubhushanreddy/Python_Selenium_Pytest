# Python_Selenium_Pytest
Sample Project developed with Selenium using Pytest Framework

Requirements:
--------------
Python 3 version
pytest
pytest-html(To fetch html report)
pytest-repeat(To run a test case multiple times)
pytest-xdist(To run test cases in Parallel)

Usage:
-------
1) Download the project
2) Switch to project folder or directory
3) Use command: pytest -s (It should run all the test cases that follows pytest test case naming conventions)
